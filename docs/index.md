# Projeto Radar Educacional

Bem-vindo à documentação do Datathon Fase 5. Este projeto visa identificar o risco de defasagem escolar dos alunos da ONG Passos Mágicos.

### Objetivo
Construir uma solução de inteligência de dados que permita à ONG agir preventivamente antes que o aluno perca o vínculo com o nível educacional adequado à sua idade.
